# Assets

This folder is reserved for public-safe visuals.

Recommended future assets:

- redacted email-flow screenshot;
- redacted Railway or hosted-worker log screenshot;
- redacted architecture walkthrough image;
- product diagram exported from the Mermaid docs.

Do not add raw customer screenshots, private provider dashboards, tokens,
booking records, Sheet IDs, phone numbers, private emails, or logs copied from
the production system.
